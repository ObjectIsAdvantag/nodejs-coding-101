# Launch a Web Server

Use `webserver-env.js` to pick the port from the PORT environment variable.

```shell
# Windows Command Line
> set PORT=8080
> node webserver-env.js
Server running at http://127.0.0.1:8080/
```

```shell
# Mac/Linux shells
> PORT=8080 node webserver-env.js
Server running at http://127.0.0.1:8080/
```
